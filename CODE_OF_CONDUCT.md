# Code of Conduct

This project adopts the Advisory Standard (`skills/advisory-standard/SKILL.md`) as its behavioral baseline for all participants — human and agent alike.

## Principles

1. **Honesty first.** Report facts accurately. Never fabricate evidence, logs, or test results.
2. **Confidentiality is absolute.** Do not expose credentials, private data, or prompt traces.
3. **Evidence over claims.** Every assertion must be backed by raw, verifiable evidence — not summaries or paraphrases.
4. **Role separation is non-negotiable.** Builders produce; Auditors judge; Prime routes. No agent approves its own work.
5. **Gate enforcement.** Evidence Gate, Implementation Gate, Audit Gate, and Risk Gate exist to protect quality. They may not be silently bypassed.

## Enforcement

Violations are routed through the Digital State governance cycle:

- **Minor** (formatting, style): Prime notes and Builder corrects.
- **Material** (fabricated evidence, gate bypass, role violation): Auditor issues REJECT. The responsible agent must remediate with raw evidence.
- **Critical** (credential leak, data exposure): Kill criteria triggered immediately per `skills/premortem-plus/SKILL.md`. All work halts until rescue actions are complete.

## Attribution

This Code of Conduct derives from the Digital State Advisory Standard and applies uniformly across all profiles and sessions.
